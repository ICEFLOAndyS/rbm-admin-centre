# RBM Domain Admin
## Responsibilities
- Accountable for the administration of a specific administrative domain within the Runbook Management (RBM) application.
- Responsible for the feedback loop with the vendor, ICEFLO Limited, including service issues and feature requests.
- Responsible for the day-to-day administration of a single domain within the RBM solution. 

## Primary Interests
- Effective setup and confiuration of the specific domain within the RBM solution
- Availability and performance of the specific domain within the RBM solution

## Challenges
- Simple setting and configuration of a domain.

## Key RBM Features
- Admin Centre
- Domain-specific adminstration features within the Admin Centre

## Before vs After
**Before:** Mutliple, inconsistent admin features and user experiences.
**After:** Controlled, role based access to the relevant domains, with a common pattern and clear user interface.

## Value Proposition
- RBM Admin Centre gives the domain admin more efficient, easier access to the domain specific admin features.
- RBM enables rapid configuration and settings for new features.
