# RBM Reference Components (Categorized)

This bundle contains the RBM Design System **Reference Components** organized by category.

## Categories
- `primitives/` — foundational UI primitives
- `containers/` — layout and containment components
- `data/` — tables, filtering, pagination, row/bulk actions
- `operational/` — execution and runbook operational UI components
- `feedback/` — dialogs, toasts, skeletons, result states
- `patterns/` — composable patterns and hooks used across components
- `providers-utils/` — required infrastructure (density + utilities)

Each component folder is self-contained and includes:
- `src/` (TypeScript/React source)
- `.css` styling files
- required RBM token/base CSS files
- `README.md`
